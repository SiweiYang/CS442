7 Q7 (posted Feb 10, due Feb 24)

Language: OCaml. Do not use mutation, except for the permitted use in the starter file.

Weight: 1.0

In this question, and the next four, you will implement Algorithm W from the slides for subsets of the datatype defined in Q6 (appropriately modified).

Implement a function typeExpr : expr -> myType that infers the type of expressions built using the following type. typeExpr should raise the exception Fail parameterized by a string error message, where the string should contain the substring "mismatch" for a type mismatch, "unbound" for an unbound identifier, and "circularity" for a failed occurs check.

type myType = T_Int | T_Bool | T_Fun of myType * myType | T_Var of int

 

type expr = Int of int

          | Bool of bool

          | Var of string

          | Fun of string * expr

          | App of expr * expr

Implement a function typeEquiv : myType -> myType -> bool that checks if its two arguments are equal modulo consistent renaming of type variables. This function is useful for your own testing, and it will be tested by Marmoset. It should be included with each of the rest of the questions, and updated as necessary.

To submit: Q7.ml.
